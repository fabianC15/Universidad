🌐 Redes de Datos

Conceptos fundamentales para ingeniería — Puertos, Sockets, Ethernet, Wi‑Fi y modos de transmisión

🔌 Puertos de red & Sockets

📡 Puertos físicos

  • RJ-45 – Ethernet estándar
  • SFP/SFP+ – Fibra óptica
  • Puerto consola – Configuración inicial

🧠 Puertos lógicos

  • 0–1023 (Conocidos): HTTP(80), HTTPS(443), SSH(22)
  • 1024–49151 (Registrados): MySQL(3306), SQL Server(1433)
  • 49152–65535 (Dinámicos): asignación temporal

🔗 Socket

Definición: IP + Puerto — Punto final de comunicación bidireccional.

  • Identificación única
  • Protocolo TCP o UDP
  • Estados: Escuchando, Establecido, Cerrado

Ejemplo: 192.168.1.10:443

📀 IEEE 802.3 · Ethernet

📌 Estándar por cable

Define redes LAN con acceso al medio (CSMA/CD).

  • 10BASE-T → 10 Mbps / UTP
  • 100BASE-TX → Fast Ethernet (100 Mbps)
  • 1000BASE-T → Gigabit (1 Gbps) Cat5e/6
  • 10GBASE-SR → 10 Gbps / fibra multimodo

⚡ Medios físicos

  • Par trenzado (UTP/STP)
  • Fibra óptica multimodo/monomodo
  • Categorías: Cat5e, Cat6, Cat6a

📡 IEEE 802.11 · Wi-Fi

📶 Banderas técnicas

  • 802.11b/g → 2.4 GHz
  • 802.11n (Wi-Fi 4) → MIMO
  • 802.11ac (Wi‑Fi 5) → 5 GHz / altas velocidades
  • 802.11ax (Wi‑Fi 6) → alta densidad + eficiencia

🌊 Ondas de radio

Bandas: 2.4 GHz, 5 GHz y 6 GHz (Wi‑Fi 6E).
